1. Basic objective is to help students to collect notes, exam question papers etc.

2. The Project will include 6  sections as:

   -Club section will contain information about all the Clubs and their activities in the college.

   -Exam section will contain all the previous year papers.

   -Notes section will contain all the subjects notes divided into modules.

3. The Project consists of basically six sections defined in a single window. 

   -These sections are named as Club Section, Exam section, Notes section, Faculty section, GLA news and last section is contact us.

   -News section will contain all the news of college Such as conferences and different workshops.  

   -Faculty section will contain about the faculties information like there cabin number and email id.

   -Last contact us section will contain about the detail of developers of the applications.
